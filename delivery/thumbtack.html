Section 5 — Key Insights from Client Survey

Source: Thumbtack AI Brief: Eng & IT (H1 2026) — shared by Marissa Scully, Apr 16

Tool Stack (by daily usage)
  • Cursor — ~57% use multiple times/day (highest)
  • Claude Code — strong daily adoption
  • ChatGPT Enterprise — regular use
  • Google Gemini — moderate
  • Atlassian AI / Rovo — lighter use
Top AI Applications
  • Coding (highest "very useful")
  • Code review, tests, technical docs
  • Debugging / troubleshooting
  • Research and information gathering
  • Summarizing information
Key Tensions to Address
  • High adoption (4.6) but low guidance on best practices (3.7)
  • Excitement about AI value (4.3) coexists with job security concern (2.5)
  • High confidence in catching bad output (4.1) but low trust in output accuracy (3.2)
  • Auth friction with Claude Code is a major pain point
What They Want
  • Move beyond experimentation → embedded in core workflow
  • Dedicated learning workstreams, not "throw darts"
  • Ship things with AI, not just try it out
  • Protected time to build proficiency
